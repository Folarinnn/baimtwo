
Requirements:
1. Need to have docker installed and running
2. You will need to create an ECR for your docker image

RUN Commands (update account-number & Region):
1. aws ecr get-login-password --region {Region} | docker login --username AWS --password-stdin {account-number}.dkr.ecr.{Region}.amazonaws.com
2. docker build -t python3.11:local -f Dockerfile.python3.11 .
3. docker tag python3.11:local {account-number}.dkr.ecr.{Region}.amazonaws.com/bedrock-agent-internet-search-langchain:latest
4. docker push {account-number}.dkr.ecr.{Region}.amazonaws.com/bedrock-agent-internet-search-langchain:latest
